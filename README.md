# somfy-motor-curtain-mqtt
DIY Motorize Curtain Somfy Control Using Esp8266 MQTT Protocol

List Item

- Esp8266 (Wemos D1 Mini)
- Optocoupler PC817
- Power Module Hi-Link 220VAC to 5VDC
- Resistor 1k 1/4 Watt
- Terminal Screw 2 Pin Hole
